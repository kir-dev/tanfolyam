JavaScript alapok

A második előadáshoz a jegyzetet megtaláljátok a Github repónkban. A jegyzet mellett találtok még pár .js fájlt is, ezekben további példákat találhattok a jegyzetben szereplő témakörökhöz.

Ezen az előadáson a következő témákról volt szó:

  • Általában a javascriptről (cél, eredet)
  • Szintaxis
  • Típusok: egyszerű és összetett (array és object)
  • Logikai kifejezések (== vs ===)
  • Függvények, scope-ok és closure-ök
  • Objektumok és függvények kapcsoalta
  • HTML és JS házasítása

Itt nem fogunk megállni a JavaScript tanulmányozásában. A workshop alatt lesz még lehetőség a nyelv más területeivel is megismerkedni. Szó lesz a prototípusokról, a DOM manipulációról és Ajaxról. Valamint röviden belenézünk a jQuery-be is.

HTML+CSS alapok

A harmadik előadáshoz a jegyzetet szintén megtaláljátok a Github repónkban. A jegyzett mellett találhattok még egy html és .css fájt, amiben az összes az előadáson érintett témát megnézhetitek.

A témák amikről szó volt:

  • A HTML nyelv
  • A dokumentum struktúrája
  • Document Object Model
  • Az alapvető elemek
  • HTML 5 és szemantikus web
  • CSS alapok
  • CSS színkezelés
  • A box model
  • A DOM manipulálása
  • Media query
